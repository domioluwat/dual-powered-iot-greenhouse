# Dual-Powered IoT Greenhouse System

This project implements an IoT-based greenhouse monitoring and control system powered by **solar energy with grid/battery backup**. The system monitors environmental conditions and controls greenhouse loads automatically or manually.

## Features
- Dual power source (solar + grid/battery)
- Temperature and humidity monitoring
- Soil moisture monitoring
- Light intensity monitoring
- Automatic and manual control modes
- Remote monitoring via IoT dashboard

## Hardware Components
- ESP32 
- 2 Solar panels
- Battery storage
- DHT11 sensor
- Soil moisture sensor
- LDR (light sensor)
- Relay modules
- Water pump, fan, bulb

## Software
- Arduino IDE
- IoT platform (e.g., Blynk)

## Power Concept
Battery is the primary power source. The system automatically switches to grid or solar power to recharge the battery.
